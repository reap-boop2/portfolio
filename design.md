# Portfolio Design System

## 设计理念

以小米 MIMO 官网为灵感，融合极简主义与动态交互，打造一个"白纸黑墨"般的个人作品集。核心理念：用最少的元素传达最强的视觉冲击，让动效本身成为内容。

## 视觉风格

### 色彩体系

| Token | 值 | 用途 |
|-------|------|------|
| `--bg` | `#fafafa` | 页面背景，接近纯白 |
| `--white` | `#ffffff` | 卡片背景 |
| `--black` | `#0a0a0a` | 主文字、黑球、按钮 |

设计原则：严格的黑白灰体系，不引入彩色。通过黑色的浓淡和留白的比例创造层次。

### 字体

- **展示字体**: `Playfair Display` (Italic) — 用于 Hero "Leo" 名字，提供优雅的衬线感
- **UI 字体**: `Space Grotesk` — 现代几何无衬线，用于标题和正文
- **中文字体**: `Noto Sans SC` — 与 Space Grotesk 搭配的中文字体
- **等宽字体**: `Space Mono` — 保留用于代码相关展示

### 间距 & 圆角

- 区域间距: `140px` 上下 padding
- 卡片圆角: `16px` (标准) / `24px` (大号)
- 按钮圆角: `100px` (胶囊) / `16px` (矩形按钮)
- 网格间距: `20px`

## 核心交互动效

### 1. 反重力悬浮系统 (Anti-Gravity Floaters)

页面背景中分布 18 个几何元素（圆环、圆点、三角形、十字），它们：

- **缓慢上浮**: 模拟反重力效果，以不同速度向上漂浮
- **正弦波摆动**: 横向和纵向的周期性摆动，模拟在气流中漂浮
- **鼠标排斥**: 鼠标移动时，元素会被轻微"推开"（通过 spring 物理引擎实现）
- **视差滚动**: 滚动时以不同深度系数移动，创造纵深感
- **自旋转**: 每个元素以随机速度缓慢旋转

参数:
```
amplitude: 8-28px (浮动幅度)
parallax: 0.02-0.08 (深度系数)
springStiffness: 0.02
springDamping: 0.92
```

### 2. Spring 物理引擎

自实现的弹簧物理系统，控制所有惯性动画：

```javascript
class Spring {
  // stiffness: 弹力系数 (越大越硬)
  // damping: 阻尼系数 (越小越弹)
  update() {
    force = (target - current) * stiffness
    velocity = (velocity + force) * damping
    current += velocity
  }
}
```

应用场景:
- **光标跟随**: stiffness=0.15, damping=0.75 → 轻快但有弹性
- **黑球跟随鼠标**: stiffness=0.03, damping=0.9 → 慵懒的延迟跟随
- **悬浮元素响应鼠标**: stiffness=0.02, damping=0.92 → 极慢的漂移

### 3. 大黑球 (Big Ball)

Hero 区域的核心视觉元素：

- 280x280px 纯黑圆形
- **鼠标跟随**: 以 spring 物理延迟跟随鼠标，制造悬浮感
- **滚动视差**: 滚动时缩小 (scale 0.6) + 上移 + 渐隐
- **文字层叠**: z-index 设置使文字在球上方，形成层次交错

### 4. 卡片 3D 倾斜 (Magnetic Tilt)

所有卡片支持鼠标跟随的 3D 倾斜效果：

```
perspective: 600px
rotateY: ±6deg (基于鼠标 X 位置)
rotateX: ±6deg (基于鼠标 Y 位置)
```

配合 `spring` easing (`cubic-bezier(0.34, 1.56, 0.64, 1)`) 实现回弹效果。

### 5. 滚动揭示动画 (Scroll Reveal)

使用 IntersectionObserver 触发：

- **默认**: translateY(50px) + opacity(0) → translateY(0) + opacity(1)
- **缓动**: `cubic-bezier(0.16, 1, 0.3, 1)` (expo out)
- **级联延迟**: `.d1`~`.d6` 递增 0.1s，形成瀑布效果
- **阈值**: 12% 可见时触发
- **单次触发**: 元素只在首次进入视口时动画

### 6. 数字计数器 (Count-Up)

技能卡片中的数字在滚动到可见时从 0 跳动到目标值：

- 持续时间: 1200ms
- 缓动: ease-out cubic (`1 - (1-t)^3`)
- 与卡片 reveal 动画同步触发

### 7. 自定义光标

- 20px 圆环跟随鼠标，使用 spring 物理产生微延迟
- Hover 状态扩大到 50px + 半透明填充
- 原生光标隐藏 (`cursor: none`)
- 移动端自动隐藏

### 8. 导航栏行为

- 固定在顶部居中，胶囊样式
- 向下滚动 > 300px 时隐藏，向上滚动时显示
- 滚动 > 60px 时增加背景模糊和阴影
- 当前 section 自动高亮 (白色背景 + 阴影)

### 9. 滚动进度条

- 固定在页面顶部的 2px 黑色进度条
- 使用 `scaleX` transform 实现平滑伸展
- 实时反映滚动位置

## 动效时序表

| 阶段 | 时间 | 动画 |
|------|------|------|
| 页面加载 | 0ms | 悬浮元素开始淡入 |
| 页面加载 | 200ms | "HELLO, I'M" 从下方滑入 |
| 页面加载 | 300ms | 副标题淡入上移 |
| 页面加载 | 500ms | "Leo" 从下方滑入 |
| 页面加载 | 1800ms | 底部提示文字淡入 |
| 滚动到达 | 0ms | section 标题 reveal |
| 滚动到达 | 100-600ms | 子元素级联 reveal |
| 鼠标移入 | 即时 | spring 弹性反馈 |

## 技术实现

- **零依赖**: 纯 HTML + CSS + Vanilla JS，无外部库
- **性能优化**: `will-change`, `requestAnimationFrame`, `passive` 事件
- **响应式**: 768px 断点，移动端隐藏光标和悬浮元素
- **物理引擎**: 自实现 Spring class，统一所有弹性动画

## 内容替换指南

需要替换的占位内容：

1. **Hero**: "Leo" → 你的名字, "AI Agent Engineer / 23岁 / 海南海口" → 你的信息
2. **技能卡片**: 图标、标题、描述、统计数字
3. **项目卡片**: 标题、标签、描述、年份
4. **合作区**: 服务类型、描述
5. **联系区**: 坐标、职业、技能标签、邮箱、GitHub、微信号
6. **导航**: "LEO" logo, "海南 海口" 位置
7. **Footer**: 名字和年份

所有文本内容集中在 HTML 中，样式完全通过 CSS class 控制，替换时只需修改 HTML 标签内的文字即可。
